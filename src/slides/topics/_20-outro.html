Kinetic CSS

Motion that moves, thinks, and breathes.
In these slides we built motion scenes with real APIs: enter transitions, timelines, paths, composition, and progressive enhancement. The goal was simple: show what CSS can do today.
  • @property and @starting-style for controllable motion
  • View timelines with animation-timeline: view()
  • View Transitions for state changes
  • Anchor positioning, popovers, and motion paths (offset-path)
  • Discrete transitions, container queries, trigonometry, 3D transforms, animation composition
  • Accessibility: prefers-reduced-motion and focus
