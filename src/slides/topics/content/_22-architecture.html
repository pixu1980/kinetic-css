Motion Architecture

The blueprint: how CSS motion APIs layer together into a coherent system.

The Stack

Foundation @property, Custom Properties, Design Tokens
Primitives translate, scale, rotate, offset-path, filter
Triggers :hover, :focus, @starting-style, scroll(), view()
Composition animation-composition, View Transitions, timeline ranges
Guardians prefers-reduced-motion, container queries, @supports
  • Foundation: typed variables for interpolation
  • Primitives: the transform and filter building blocks
  • Triggers: what starts the motion
  • Composition: layering and sequencing
  • Guardians: accessibility and progressive enhancement
