Motion Patterns

Reusable recipes: stagger, reveal, micro-interactions, state transitions.

The Essentials

Staggered Entrance 

.item {
  animation-delay: calc(var(--i) * 80ms);
}
Scroll Reveal 

.reveal {
  animation-timeline: view();
  animation-range: entry 0% entry 100%;
}
Hover Lift 

.card:hover {
  translate: 0 -8px;
  box-shadow: 0 12px 24px oklch(0 0 0 / 0.2);
}
State Morph 

::view-transition-old(card),
::view-transition-new(card) {
  animation-duration: 0.3s;
}
  • These four patterns cover 80% of UI motion needs
  • Combine them for richer choreography
  • Always test with reduced motion enabled
  • Use custom properties for easy theming
