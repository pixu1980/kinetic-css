Motion without jank. Know what triggers layout, paint, and composite.

The Golden Rule

Compositor Only
  • transform
  • opacity
  • filter (GPU)
  • clip-path (simple)
60fps
Paint
  • background
  • color
  • box-shadow
  • border-radius
Careful
Layout
  • width, height
  • margin, padding
  • top, left
  • font-size
Avoid 
    
    /* Use will-change sparingly */
    .animated-element {
      will-change: transform, opacity;
    }

    /* Remove after animation */
    .animated-element.done {
      will-change: auto;
    }
  • Stick to transform and opacity for silky animations
  • will-change hints the browser but overuse hurts memory
  • Use DevTools Performance panel to profile
  • content-visibility can help with off-screen elements
