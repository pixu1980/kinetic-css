3D Transforms

Use perspective and preserve-3d to create depth.

Demo

    
    .deep-card-container { perspective: 1000px; }

    .deep-card {
      transform-style: preserve-3d;
    }

    .deep-card:hover,
    .deep-card:focus-within {
      transform: rotateY(15deg) rotateX(-10deg) translateZ(50px);
    }
