Stagger Math

Each element knows its index and phases in at the right moment, like a choir on cue.

Demo

    
    /* Progressive enhancement: sibling-index() */
    .logo-stagger {
      --i: sibling-index();
      animation-delay: calc(var(--i) * 100ms);
    }

    /* Fallback: :nth-child */
    .logo-stagger:nth-child(1) { --i: 0; }
    .logo-stagger:nth-child(2) { --i: 1; }
    /* ... */
  • sibling-index() and sibling-count() auto-compute position
  • Each element knows where it sits in the sequence
  • Fallback to :nth-child for browsers without support
  • API focus: sibling-index(), sibling-count(), :nth-child fallback
