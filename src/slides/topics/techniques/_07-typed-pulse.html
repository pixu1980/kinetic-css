Typed Pulse

The logo pulses like an electronic heartbeat: controlled breathing via @property.

Demo

    
    @property --scale {
      syntax: '<number>';
      inherits: false;
      initial-value: 1;
    }

    .logo-property {
      animation: property-dance 3s ease-in-out infinite;
      transform: scale(var(--scale)) rotate(var(--rotate));
    }
  • The pulse is not a simple scale, it is a controlled breath: smooth, fluid, always the same
  • @property with syntax number makes --t interpolate between 0 and 1
  • The keyframes animate --t, and scale is computed from it
  • API focus: @property typed custom property, keyframes on custom property
