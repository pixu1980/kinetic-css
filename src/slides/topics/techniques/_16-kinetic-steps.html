Kinetic Steps

The logo moves in controlled, mechanical steps. Each segment refreshes intentionally.

Demo

steps(5)
steps(10)
linear() 
    
    .logo-steps {
      animation: slide-in 1.5s steps(var(--steps, 8)) infinite;
    }

    .logo-linear {
      animation: slide-in 1.5s linear(0, 0.8, 0.9, 1) infinite;
    }

    @keyframes slide-in {
      from { translate: -100% 0; opacity: 0; }
      to   { translate: 0 0; opacity: 1; }
    }
  • steps() creates discrete, mechanical motion
  • Great for typewriter effects and retro aesthetics
  • linear() allows custom easing curves
  • API focus: steps(), linear() easing functions
