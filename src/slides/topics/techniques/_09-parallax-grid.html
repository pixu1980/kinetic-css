Parallax Grid

Elements move at different depths: small logos scroll slower, creating parallax.

Demo

Scroll the grid to animate the hero logo. 
    
    .timeline-container { timeline-scope: --grid-timeline; }
    .timeline-scroller { scroll-timeline: --grid-timeline block; }
    .hero-timeline-bound { animation-timeline: --grid-timeline; }
  • Small logos move faster as you scroll, hero logo stays more stable
  • This creates depth without 3D transforms
  • timeline-scope hoists the scroll-timeline to an ancestor
  • API focus: scroll-timeline, animation-timeline, timeline-scope
