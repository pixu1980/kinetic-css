Container Queries

Adapt animation density based on the component size, not the viewport size.

Demo

Wide container (fast)
Narrow container (slow) 
    
    @container logos-grid (width < 400px) {
      .logo-cq { animation-duration: 1s; }
    }

    @container logos-grid (width >= 700px) {
      .logo-cq { animation-duration: 0.3s; }
    }
