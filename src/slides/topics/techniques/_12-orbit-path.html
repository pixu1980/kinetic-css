Orbit Path

The logo orbits around a point, rotating to align with the curve like a satellite.

Demo

Kinetic CSS

    
    .orbit-logo {
      offset-path: ellipse(220px 100px at 50% 50%);
      animation: orbit-path 8s linear infinite;
    }

    @keyframes orbit-path {
      to { offset-distance: 100%; }
    }
  • The logo follows an elliptical orbit around the title
  • offset-rotate auto aligns the element to the path direction
  • This is pure CSS, no JavaScript animation loop needed
  • API focus: offset-path, offset-distance, offset-rotate
