Additive Motion

Two motions coexist: slow rotation plus breathing scale. Organic, not mechanical.

    @keyframes spin { to { rotate: 360deg; } }
    @keyframes breathe { 50% { scale: 1.15; } }

    .logo-additive {
      animation:
        spin 6s linear infinite,
        breathe 2s ease-in-out infinite;
      animation-composition: add;
    }
  • Spin provides continuous slow rotation
  • Breathe adds a scale pulse on top
  • animation-composition: add combines them without overwriting
  • API focus: animation-composition, multiple animations
