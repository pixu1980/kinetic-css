Glass Blur Entrance

The logo passes behind frosted glass. A blur that gradually focuses, like a memory sharpening.

Demo

    
    .glass-overlay {
      backdrop-filter: blur(20px);
      animation: glass-clear 2s ease-out forwards;
    }

    @keyframes glass-clear {
      from { backdrop-filter: blur(20px); opacity: 1; }
      to   { backdrop-filter: blur(0); opacity: 0; }
    }

    .logo-glass {
      filter: blur(10px);
      animation: focus-in 2s ease-out forwards;
    }

    @keyframes focus-in {
      from { filter: blur(10px); }
      to   { filter: blur(0); }
    }
  • filter: blur() applies to the element itself
  • backdrop-filter: blur() affects what's behind
  • Combine both for a layered frosted glass effect
  • API focus: filter, backdrop-filter, blur()
