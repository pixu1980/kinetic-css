    .tooltip-popover {
      position-anchor: --trigger;
      top: anchor(bottom);
      left: anchor(center);
      transition: opacity 0.3s, transform 0.3s;

      @starting-style {
        opacity: 0;
        transform: translateY(-10px) rotateX(15deg);
      }
    }
  • The tooltip flips in gently, like a tag being attached
  • Popover API provides the show/hide behavior natively
  • @starting-style handles the entrance animation
  • API focus: Popover API, anchor positioning, @starting-style
