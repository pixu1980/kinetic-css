Morph Switch

The grid transforms into a detail view like a film cut. No fade, just a scene change.

Demo

    
    document.startViewTransition(() => {
      // Toggle the DOM state
    });
  • This is not a fade, it is a morphing scene change
  • startViewTransition captures the old state, then applies the new state
  • The browser interpolates between them automatically
  • API focus: View Transitions API, ::view-transition-old/new pseudos
