Trigonometry

Use sin() and cos() for organic motion with phase offsets.

Demo

    
    .logo-trig {
      --phase: calc(var(--i, 0) * 0.5);
      transform:
        translateY(calc(sin(var(--wave-offset) * 3.14159 + var(--phase)) * 30px))
        scale(calc(1 + cos(var(--wave-offset) * 3.14159 + var(--phase)) * 0.1));
    }
