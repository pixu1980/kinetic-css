Context Choreo

Choreography adapts to the container. Narrow space means compact motion, wide space allows broader arcs.

Demo

Narrow (compact motion)
Wide (broader arc) 
    
    .cq-container { container-type: inline-size; }

    @container (width < 350px) {
      .logo-cq { animation-duration: 2s; }
    }

    @container (width >= 400px) {
      .logo-cq { animation-duration: 0.8s; }
    }
  • The motion adapts to available space, not viewport size
  • Narrow containers get slower, more compact animations
  • Wide containers allow faster, more expressive motion
  • API focus: @container, container-type
