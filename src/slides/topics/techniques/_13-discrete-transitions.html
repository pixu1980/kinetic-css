Discrete Transitions

Animate in and out a panel that uses display via transition-behavior: allow-discrete.

Demo

Panel 2

    .discrete-panel {
      transition:
        opacity 0.4s ease,
        transform 0.4s ease,
        display 0.4s allow-discrete;

      @starting-style {
        opacity: 0;
        transform: translateX(-30px);
      }
    }

    .discrete-panel[hidden] {
      display: none;
      opacity: 0;
      transform: translateX(30px);
    }
