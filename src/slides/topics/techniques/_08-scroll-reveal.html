Scroll Reveal

When the element enters the viewport, it emerges like from behind frosted glass.

Demo

    
    .logo-view-timeline {
      animation: reveal-in linear both;
      animation-timeline: view();
      animation-range: entry 0% entry 100%;
    }
  • The logo starts transparent and slightly shifted, then takes form as you scroll
  • view() creates a timeline based on the element visibility in the viewport
  • animation-range controls when the animation starts and ends
  • API focus: animation-timeline: view(), animation-range
