Magnetic Snap

The logo moves like attracted to a magnet: accelerates, overshoots, then bounces back softly.

Demo

    
    .logo-magnetic {
      transition:
        translate 0.4s var(--ease-overshoot),
        scale 0.4s var(--ease-overshoot);
    }

    .logo-magnetic:hover,
    .logo-magnetic:focus-visible {
      translate: 0 -12px;
      scale: 1.12;
    }
  • The logo accelerates toward the target point, overshoots slightly, then settles back
  • We use a cubic-bezier with values greater than 1 to create the overshoot
  • The bounce is soft, not mechanical, like a magnetic pull
  • API focus: transitions, cubic-bezier, individual translate and scale
