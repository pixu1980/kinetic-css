Individual Transforms

Use rotate, scale, and translate as first class properties. Hover or focus a logo.

Demo

    
    .logo-transform {
      transition: rotate 0.3s ease-out,
        scale 0.3s ease-out,
        translate 0.3s ease-out;
    }

    .logo-transform:hover,
    .logo-transform:focus-visible {
      rotate: 5deg;
      scale: 1.15;
      translate: 0 -8px;
    }
