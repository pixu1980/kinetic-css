Sponsors

A staggered entrance with @starting-style and a simple per-item delay.

Demo

    
    .sponsor-enter {
      transition-delay: calc(var(--i, 0) * 80ms);

      @starting-style {
        opacity: 0;
        transform: translateY(30px) scale(0.8);
      }
    }
  • Roll call: logos arrive in waves, like an impulse traveling left to right
  • Each logo has a micro tilt and settle, so the grid feels alive but not chaotic
  • The stagger uses --i for delay calculation: 80ms between each logo
  • @starting-style handles the initial hidden state
  • API focus: grid layout, :nth-child for stagger, @starting-style
