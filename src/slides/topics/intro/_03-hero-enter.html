@starting-style

Entry animations without JavaScript. The element defines a starting style, then transitions to its final state.

Demo

    
    .hero-logo {
      opacity: 1;
      transform: scale(1) translateY(0);
      transition: opacity 0.8s ease-out,
        transform 0.8s cubic-bezier(0.34, 1.56, 0.64, 1);

      @starting-style {
        opacity: 0;
        transform: scale(0.5) translateY(40px);
      }
    }
The animation plays on element creation when the slide becomes active.
